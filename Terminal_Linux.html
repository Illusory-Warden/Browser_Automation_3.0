This command adds Microsoft's official repository to your Parrot OS so you can install Microsoft Edge.
Here‚Äôs a quick breakdown:
Download & convert Microsoft‚Äôs GPG key (for package verification).

üí° Allows you to install & update Microsoft Edge on Parrot OS! üî•

sudo rm microsoft.gpg
